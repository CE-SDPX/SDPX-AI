## Rubric การให้คะแนน (5 คะแนน)

| เกณฑ์ | 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|---|
| **App deploy ได้จริง** | URL ไม่ทำงาน | URL ทำงานแต่ error | เปิดได้แต่ไม่มีเนื้อหา | เปิดได้มีเนื้อหา | เปิดได้ สวยงาม มีเนื้อหาชัดเจน |
| **Domain มีความชัดเจน** | ไม่ชัดว่าทำอะไร | พอเข้าใจแต่คลุมเครือ | เข้าใจได้ | ชัดเจน มี user ชัด | ชัดเจน มี user + pain point ที่แก้ |
| **Team ทุกคน setup ได้** | มีแค่คนเดียวที่ทำได้ | 2 คน | 3 คน | 4 คน | ทุกคน (5 คน) |
| **README ครบถ้วน** | ไม่มี README | มี README แต่ว่างเปล่า | มีแค่ชื่อ project | มี description + how to run | ครบทุกหัวข้อ + team members |
| **การนำเสนอ** | ไม่ได้ present | present ไม่ครบ | present ครบแต่ไม่ชัด | present ชัดเจน | present ชัดเจน ตอบคำถามได้ |

**คะแนนเต็ม: 5 คะแนน** (เฉลี่ยจาก 5 เกณฑ์)

# Self-Learning: เตรียมก่อนเรียน Performance & Observability

## สิ่งที่ต้องศึกษา

### k6 Load Testing
- [video] k6 Setup for Load Testing — Step by Step Tutorial
  (https://www.youtube.com/watch?v=XR2MAivt-9E)
- [video] Basics of Load Testing with k6 and Grafana in 20 Minutes
  (https://www.youtube.com/watch?v=gvounvDSDGg)
- [reading] k6 Getting Started
  (https://k6.io/docs/get-started/running-k6/)

**จุดที่ต้องเข้าใจ:**
- Virtual Users (VUs) คืออะไร
- p95 latency ต่างจาก average อย่างไร และทำไมสำคัญกว่า
- Load test vs Stress test vs Spike test ต่างกันอย่างไร

### Observability
- [reading] The Three Pillars of Observability — O'Reilly
  (https://www.oreilly.com/library/view/distributed-systems-observability/9781492033431/ch04.html)

**จุดที่ต้องเข้าใจ:**
- Logs, Metrics, Traces ต่างกันอย่างไร
- Structured logging คืออะไร และดีกว่า `console.log` อย่างไร
